목욕 제공 확인

계획서와 상담일지 급여제공반영 내용을 기준으로 목욕 기록이 맞는지 확인합니다.

💡 목욕 확인 기준 안내
  • 급여 수립 기본 대상: 계획서에 몸씻기 도움이 있고 상담일지에서 제외되지 않은 사람 → 주차별 목욕 기록 또는 목욕거부 내역이 입력되어 있어야 정상입니다.
  • 상담일지 제외 반영: 상담일지 기록에서 목욕 제외가 공식 반영된 사람 → 해당 반영일 이후 주차에는 목욕 기록이 없어야 정상으로 필터링합니다.
  • 오류 판단 기준: 계획서상에 몸씻기 도움이 없거나 상담일지로 제외가 확정된 수급자에게 목욕 기록이 남아있는 경우 오류로 판정되어 표출됩니다.

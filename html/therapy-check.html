물리치료 제공 확인

계획서와 상담일지 반영 내용을 기준으로 주차별 물리치료 제공 기록을 확인합니다.

💡 물리치료 확인 기준 안내
  • 계획서 기준: 계획서에 물리치료(기능회복훈련)가 있는 사람 → 해당 주차에 물리치료 기록이 있어야 정상
  • 상담일지 기준: 상담일지에서 물리치료 제외가 반영된 사람 → 반영일 이후에는 물리치료 기록이 없어야 정상
  • 계획서에 물리치료가 없거나 상담일지로 제외된 사람에게 기록이 있으면 오류로 표시됩니다.
